Introduction to Unix

Network Commands

Command Description
ipconfig Displays the network configuration of the computer (Windows only)
ping 10.10.54.75 Sends test packets to a server to check if it’s alive
telnet 10.10.54.75 Connects to a remote server for terminal access

Unix Command Syntax

$ command [option] [arguments]
$ ls –sCF /usr/bin
Element Description
$Prompt symbol
lsCommand
-sCFOptions (equivalent to -s -C -F)
/usr/binArgument (target directory)
[ ... ]Optional part

Usage: $ man ls - Shows the manual documentation for the ls command.

Basic Commands

Command Description
uname -aShows system info (OS, kernel, host)
datePrints current date and time
idDisplays user ID, group ID, and groups
groupsLists groups the user belongs to
whoShows logged-in users
who \| grep unix203Filters for user unix203 in who output
finger [id]Displays detailed user info (real name, login time, etc.)
wShows users and their current activity (processes)

Terminal Setting

Command Description
ttyShows your current terminal device (e.g., /dev/pts/1)
stty -aDisplays current terminal settings (like erase, intr)
stty erase [key]Sets backspace/delete key (e.g., ^H or ^?)
echo hello | write unix203 pts/34Sends "hello" to user unix203 on terminal pts/34
