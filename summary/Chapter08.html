Regular Expressions & Unix Utilities

Regular Expression

Regular expressions represent patterns of text using metacharacters.

Meta Characters

Meta CharacterDescription
.Matches any single character
[]Matches any one of the characters inside the brackets
*Matches zero or more occurrences of the previous character
[^...]Matches any character not listed inside brackets
^Matches the beginning of the line
$Matches the end of the line
\Escapes a meta character
?Matches zero or one of the preceding character
[a-z]Matches lowercase a to z
^.*Matches from start with any characters
^.*[\[\]]Matches any line starting with any chars and containing [ or ]

U.S. Telephone Number Pattern

[0-9][0-9][0-9]\.[0-9][0-9][0-9]\.[0-9][0-9][0-9][0-9]

This pattern matches: 123.456.7890
