Stopping Processes

CommandDescription
Ctrl + CStop foreground process
killSend signal to process
kill -lList signal names
kill -9 PIDForce kill (SIGKILL)
pkill program_nameKill process by name

A virtual filesystem containing runtime system info.

TermDescription

BackgroundRuns in background; frees terminal
