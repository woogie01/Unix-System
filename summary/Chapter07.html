Chapter 07

Vi

Moving within a File

KeyDescription
hMove left
jMove down
kMove up
lMove right
0 or ^Move to the beginning of the line
$Move to the end of the line
EnterMove to the next line
Ctrl + fMove forward one full page
Ctrl + bMove backward one full page
Ctrl + dMove down half a page
Ctrl + uMove up half a page
:0, :1Go to the first line
:$Go to the last line
:100Go to the 100th line
Ctrl + %Jump to matching { or }

Searching in vi or vim

CommandDescription
/stringSearch forward
?stringSearch backward
nNext match in same direction
NPrevious match (reverse direction)

Exit and Save

CommandDescription
:qQuit (ask to save if modified)
:q!Force quit without saving
:wWrite current file
:w!Force write
:wqWrite and quit
:wq!Overwrite and quit
ZZSave and exit (if modified)
:xSave and exit (like :wq)
:w filenameSave to another file
:r filenameInsert contents of file

Insert Mode (vim)

CommandDescription
iInsert at cursor
aAppend after cursor
oOpen new line below
[Esc]Exit insert mode

Delete Commands

CommandDescription
xDelete 1 character
5xDelete 5 characters
ddDelete current line
3ddDelete 3 lines
DDelete to end of line
JJoin with next line
i[Enter][Esc]Insert line break

Change Commands

CommandExampleDescription
rraReplace 1 character with 'a'
3r3ra[esc]Replace 3 characters with 'aaa'
cwcwabc[esc]Change 1 word to 'abc'
3cw3cwabc[esc]Change 3 words to 'abc'
cwab cd 12[esc]-Change current word to 'ab cd 12'
3cwab cd 12[esc]-Change 3 words to 'ab cd 12'

Undo / Redo

KeyDescription
uUndo last change
uuUndo the undo
.Repeat last change

Example Flow

/abc
cwABCD[esc]
n.
n.

Copy and Move

CommandAction
yyCopy 1 line
3yyCopy 3 lines
ddCut 1 line
3ddCut 3 lines
pPaste below

Running Commands

CommandFunction
:%s/abc/ABCD/gReplace all 'abc' with 'ABCD'
:10,$ s/abc/ABCD/gReplace 'abc' from line 10 to end
:0,$ s/^M//gRemove all ^M characters (Ctrl+V Ctrl+M)
:100,200 w! filenameWrite lines 100 to 200 to filename
